# 用户使用流程

## 1. 总体流程

```mermaid
flowchart TD
    A[用户上传小说文本] --> B[系统解析章节]
    B --> C[提取人物/地点/事件]
    C --> D[构建 Story Graph]
    D --> E[生成角色记忆]
    E --> F[规划剧本场景]
    F --> G[生成对白/动作/旁白]
    G --> H[输出 YAML 剧本]
    H --> I[质量评估与修改建议]
    I --> J[作者编辑与导出]
```

## 2. 演示界面功能说明

> 演示界面基于 Gradio 实现，聚焦核心转换流程的可交互展示，不是完整前端产品。

| 模块 | 功能 |
|---|---|
| 输入区 | 粘贴或上传 txt / md 小说文件，点击开始转换 |
| 解析结果区 | 展示识别的人物、地点、事件列表 |
| 剧本输出区 | 展示生成的 YAML 剧本内容，支持复制下载 |
| 质量评估区 | 展示各维度评分和修改建议 |
